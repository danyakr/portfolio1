## Тема 4
### Использование системы компьютерной алгебры при работе с матрицами

#### Лабораторная работа
- [ЛР 4.1](https://www.google.com/url?q=https%3A%2F%2F1drv.ms%2Fu%2Fs!Aj9eLzWpyS4ogindLB5fO88J-kdg%3Fe%3DPYfHob&sa=D&sntz=1&usg=AOvVaw3ZNQ-jlQOMNESsMsrGB22B)

Особенности и способы выполнения действий с матрицами.  
Для вычисления произведения матриц в системе компьютерной алгебры Maxima нужно записать матрицы через ".".  
Для нахождения квадрата/куба матрицы нужно написать имя матрицы через "." столько раз, какова степень, в которую возводится матрица.  
Для вычисления матричного многочлена необходимо пользоваться пунктами 1 и 2 и умножением матрицы на число (для этого нужно написать имя матрицы и через "*" умножить на необходимый коэффициент).

- [ЛР 4.2](https://www.google.com/url?q=https%3A%2F%2F1drv.ms%2Fu%2Fs!Aj9eLzWpyS4ogix3mCrRsy1ntDR6%3Fe%3DaBr6bg&sa=D&sntz=1&usg=AOvVaw3IkyfZelEgNImeuU5ktX3v)

Ход работы и способы выполнения транспонирования матриц, нахождения обратной матрицы.  
Для получения транспонированной матрицы из исходной можно либо воспользоваться главным меню «Алгебра – Транспонировать матрицу», либо с помощью функции `transpose`.  
Для нахождения обратной матрицы сначала необходимо убедиться, что определитель не равен нулю. Это можно сделать двумя способами:

1. При помощи команды главного меню: «Алгебра – Определитель».  
2. Ввод команды `determinant`.  

Для нахождения обратной матрицы существует три способа:

1. При помощи команды главного меню: «Алгебра – Обратить матрицу».  
2. Ввод команды `invert`.  
3. Возведение матрицы в степень (–1).

- [ЛР 4.3](https://www.google.com/url?q=https%3A%2F%2F1drv.ms%2Fu%2Fs!Aj9eLzWpyS4ogipj21SENNVPdTHj%3Fe%3DEqhR0u&sa=D&sntz=1&usg=AOvVaw0s-sn7tdLXQfpSdDKjVPwq)

Способы нахождения ступенчатой матрицы:  
1. При помощи функции `triangularize` (не нормирует элементы главной диагонали).  
2. При помощи функции `echelon` (нормирует элементы главной диагонали — все элементы равны 1).

Для вычисления ранга матрицы используют команду `rank`.

- [ЛР 4.4](https://www.google.com/url?q=https%3A%2F%2F1drv.ms%2Fu%2Fs!Aj9eLzWpyS4ogit9m8zBmy6cfjv4%3Fe%3DJJcb9m&sa=D&sntz=1&usg=AOvVaw1Eh5VYFWsqeQCCNf4WY4Qg)

Чтобы из исходной матрицы получить новую, удалив одну или несколько строк/столбцов, используют команду `submatrix(x, M, y)`.

## Тема 5
### Использование системы компьютерной алгебры при работе с выражениями и уравнениями

#### Лабораторная работа

- [ЛР 5.0](https://www.google.com/url?q=https%3A%2F%2F1drv.ms%2Fu%2Fs!Aj9eLzWpyS4oglDXBUT_5iSf3HOI%3Fe%3DIcKMOd&sa=D&sntz=1&usg=AOvVaw3Y6LZXyKNKMhqNi7h6717w)

В этой части изучено:

Флаг numer

Приведении конечной десятичной записи чисел к рациональной. Флаг keepfloat

Функция rat(число) приводит число, записанное в виде конечной десятичной дроби, к рациональному числу, записанному обыкновенной дробью.

- [ЛР 5.1](https://www.google.com/url?q=https%3A%2F%2F1drv.ms%2Fu%2Fs!Aj9eLzWpyS4oglH6dzVKxI1Ne6gE%3Fe%3D4z3z35&sa=D&sntz=1&usg=AOvVaw0fBXL_2l3_cGfkeDgAW7vl)

Функция rat - Преобразование рационального выражения к канонической форме (Canonical Rational Expression, CRE)

Если каноническая форма не нужна, то есть надо оставить общий вид, тогда применяют функцию ratdisrep(выражение).

Если необходимо представить выражение в виде суммы простейших дробей, то эту задачу решает функция partfrac(выражение, имя переменной).

За раскрытие скобок отвечает функция ratexpand(выражение).
